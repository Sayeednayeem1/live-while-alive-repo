Search Engine

Google is a world wide web network. It's basically a search engine. Google Search is a fully-automated search engine that uses software known as web crawlers that explore the web regularly to find pages to add to our index. Google has the capability to calculate a site's relevance according to the number of pages and links pointing back to the original site. The success of Google has led to it being the most-used search engine in the U.S. market, managing over 3 billion pages a day.
